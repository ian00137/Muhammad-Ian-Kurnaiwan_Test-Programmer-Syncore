  2. Jelaskan mengenai fungsi dan penggunaan if else, while, while do, switch case, for, foreach ?
    3. Jawab:
    • If Else Merupakan sebuah kondisi atau percabangan, dimana blok kode yang ada didalam if akan dijalankan jika kondisinya terpenusi, dan else dijalankan jika if nya tidak terpenuhi, else sendiri opsional dan tidak harus selalu ada.
    • While Merupakan sebuah perulangan dimana kode dalam while tersebut akan terus menerus di ulang selama kondisinya masih terpenuhi.
    • While Do Hampir sama seperti while, namun while do akan menjalankan kode sebelum mengecek kondisinya, jadi meskipun kondisi tidak sesuai, kodingan akan tetap dijalankan sekali.
    • Switch Case Merupakan sebuah percabangan, hampir sama seperti if else, dimana akan menjalankan case seusai dengan nilai sebuah variabel, switch case juga memiliki default yang akan dijalankan jika nilai variabel dan case tidak ada yang sesuai.
    • For Merupakan sebuah perulangan seperti while, namun sedikit lebih kompleks diamana for memiliki inisialisasi index, kondisi index, dan perubahan nilai index.
    • ForeachMerupakan sebuah perulangan, dimana jumlah perulangan akan menyesuaikan dengan jumlah index di sebuah variabel bertipe data collection seperti array sehingga lebih flexibel.
